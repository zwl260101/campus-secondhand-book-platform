# \# CampusBookTrade - 校园二手书交易平台

# 

# \## 项目简介

# 这是一个基于 Python 开发的轻量级命令行后端系统，旨在为校园学生提供便捷的二手书信息发布与检索服务。

# 

# \## 核心功能

# \- \*\*书籍管理\*\*：支持书籍发布、信息持久化存储及下架处理。

# \- \*\*搜索算法\*\*：基于 SQL 模糊查询实现多维度（书名/作者）关键词检索。

# \- \*\*异常处理\*\*：全链路捕获非法输入，确保程序健壮性。

# 

# \## 技术栈

# \- \*\*语言\*\*：Python 3.x

# \- \*\*数据库\*\*：SQLite3

# \- \*\*架构\*\*：面向对象编程 (OOP)

# 

# \## 如何运行

# 1\. 克隆代码到本地。

# 2\. 运行 `python main.py` 启动系统。
